SoftEther

サーバー

Arch Linux

インストール

$ yaourt -Sb softethervpn

次のコマンドがインストールされる

/usr/bin/vpnserver

/usr/bin/vpncmd

サーバーのスタート

# systemctl start softethervpn-server

# systemctl status softethervpn-server

サーバーの設定

$ vpncmd

CentOS

gcc が必要

softether-vpnserver-v4.20-9608-rtm-2016.04.17-linux-x64-64bit.tar.gz

解凍して、

make

次のコマンドができる

vpnserver

vpncmd
