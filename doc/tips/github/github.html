github の使い方

The first time copy

$ git clone https://github.com/ekzemplaro/data_base_language

To update the data

cd data_base_language

$ git add --all

$ git commit -m "commit Jan/29/2015"

$ git push -u origin master

Client

To synchronize data

cd data_base_language

$ git pull

Return
