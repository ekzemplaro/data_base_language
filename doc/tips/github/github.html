GitHub の使い方

The first time copy

$ git clone https://github.com/ekzemplaro/data_base_language

To update the data

$ cd data_base_language

$ git add --all

$ git commit -m "commit Jan/29/2015"

$ git push -u origin master

Client

To synchronize data

$ cd data_base_language

$ git pull

To force synchronization

$ git fetch origin master

$ git reset --hard origin/master

To create a repository

$ echo "# ekimae_ryokuka" >> README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git remote add originhttps://github.com/ekzemplaro/ekimae_ryokuka.git
$ git push -u origin master
ブランチのダウンロード

$ git clone -b develop https://product.backlog.jp/git/NEWS_AAA/ABBB.git

ブランチの確認

$ git branch
* develop
エラーへの対応

server certificate verification failed. CAfile: /etc/ssl/certs/ca-certificates.crt CRLfile: none

対策

export GIT_SSL_NO_VERIFY=1
