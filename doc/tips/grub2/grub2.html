Grub2 の設定

GUI は、grub-customizer　がある。

インストール方法

# add-apt-repository ppa:danielrichter2007/grub-customizer
# apt-get update
# apt-get install grub-customizer

/etc/default/grub を編集。

GRUB_DEFAULT="Ubuntu"
を変更する。

OS名を調べるには、

grep menuentry /boot/grub/grub.cfg

変更内容を反映

sudo update-grub
