msmtp

msmtp を使ったメールの送信方法

msmtp -C setting_gmail.txt -t < in01.txt

in01.txt

	From: aaa@gmail.com
	To: bbb@example.com
	Subject: Test using msmtp from gmail on Jan/23/2017
	Test
	From Gmail
	Good Morning
	Bye
setting_gmail.txt

account default
host	smtp.gmail.com
port	587
user	aaa
password	*****
#
auth	on
tls	on
tls_certcheck	off
#
from	aaa@gmail.com
#
logfile	msmtp.log
#
