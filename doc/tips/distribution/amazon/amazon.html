Amazon Linux AMI

ssh でアクセス

$ ssh -i key_pair.pem ec2-user@35.162.*.*

Amazon EC2 コンソールを使って、
セキュリティーグループで、インバウンドのIP を設定する必要がある。

バージョン確認方法
$ cat /etc/os-release
NAME="Amazon Linux AMI"
VERSION="2016.09"
ID="amzn"
ID_LIKE="rhel fedora"
VERSION_ID="2016.09"
PRETTY_NAME="Amazon Linux AMI 2016.09"
ANSI_COLOR="0;33"
CPE_NAME="cpe:/o:amazon:linux:2016.09:ga"
HOME_URL="http://aws.amazon.com/amazon-linux-ami/"
