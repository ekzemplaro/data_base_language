Ubuntu のクライアント設定

# apt-get install ldap-utils

# apt-get install libpam-ldap nscd

  1. LDAPサーバーのURIを指定
    2. base dc=example,dc=com

  2. 識別名を指定
    3. rootbinddn cn=Manager,dc=example,dc=com

  3. LDAPバージョンを指定
    4. ldap_version 3

/etc/ldap.conf に入力したデータが入る

/etc/nsswitch.conf
passwd: compat ldap
group: compat ldap
shadow: compat ldap

netgroup: ldap

/etc/pam.d/common-session
session required pam_mkhomedir.so skel=/etc/skel umask=0022
# systemctl restart nscd
