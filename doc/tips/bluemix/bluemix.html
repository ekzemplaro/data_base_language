Bluemix

bx

インストール

$ yaourt -Sb bluemix-cli

バージョン

$ bx --version
bx version 0.5.5+87df0e6-2017-07-03T05:55:45+00:00
環境の表示

$ bx config --list
Name                                  Value   
Color enabled                         -   
HTTP timeout (Second)                 0   
Trace                                 -   
User locale                           -   
Check CLI version enabled             true   
Collecting usage statistics enabled   false
環境の設定

$ bluemix config --usage-stats-collect false

アプリケーションの表示

$ bx app list

アプリケーションの停止

$ bx app stop app_ver1

使い方

$ bx api https://api.ng.bluemix.net
$ bx login -u user@example.ne.jp -p password -o example -s us
$ bx app push

cf

インストール

$ yaourt -Sb cloudfoundry-cli

Ubuntu 17.04

$ wget -q -O - https://packages.cloudfoundry.org/debian/cli.cloudfoundry.org.key | sudo apt-key add -
$ echo "deb http://packages.cloudfoundry.org/debian stable main" | sudo tee /etc/apt/sources.list.d/cloudfoundry-cli.list
# apt update
# apt install cf-cli
バージョン

$ cf --version
cf version 6.27.0+d26b32d.2017-06-08
使い方

$ cf api https://api.ng.bluemix.net
$ cf login -u user@example.ne.jp -p password -o example -s us
$ cf push
