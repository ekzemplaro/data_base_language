鍵の作成

$ cd $HOME/.ssh

$ ssh-keygen -t rsa -N "" -f ./ファイル名

$ ssh-keygen -t rsa -N "" -f key_bbb

key_bbb.pub が作成される。
これを、サーバー へ送り
$ cat key_bbb.pub >> $HOME/.ssh/authorized_keys
