Mount

iso イメージのマウント

$ sudo mount -o loop -t iso9660 ubuntu-13.10-desktop-amd64.iso /mnt/iso

NFS マウント

# apt-get install nfs-common

$ sudo mount server_aa:/var/www /mnt/server_aa/www

ftp マウント

$ sudo curlftpfs -o uid=1000,gid=1000,user=scott:tiger ftp://server_aa /mnt/ftp

$ sudo curlftpfs -o uid=1000,gid=1000,codepage=cp932,user=scott:tiger ftp://server_aa /mnt/ftp

パーティションのマウント

$ sudo mount /dev/sda8 /mnt/sda8

ハードディスクの UUID の調べ方

ls -l /dev/disk/by-uuid

sudo blkid -p /dev/sda1
