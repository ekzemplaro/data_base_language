sudoers & group

sudoers の追加 in Ubuntu 14.04

$ sudo gpasswd -a ユーザー名 sudo

ユーザーが所属しているグループを確認
$ id ユーザー名

$ id uchida
uid=1000(uchida) gid=1000(uchida) groups=1000(uchida),10(wheel),142(docker)
グループにユーザーを追加
# usermod -G グループ名 ユーザー名
# usermod -G testgrp uchida

グループを追加
# groupadd グループ名
# groupadd testgrp

Arch Linux

/etc/sudoers
