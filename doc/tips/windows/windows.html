Windows コマンド の使い方

フォルダーのアクセス権設定を調べる

icacls folder

icacls \\server\folder

ユーザーのグループを調べる

net user user_name /domain

グループのメンバーを調べる

net group group_name /domain

英語、日本語表示の切り替え

chcp 437

chcp 932

chcp

ネットワークのリセット

ipconfig /release
ipconfig /renew
ドライブのマウント

マウント状況の確認

net use

マウント

net use z: \\host_name\share

net use z: \\host_name\share password /user:domainname\username

切断

net use z: /delete

シンボリックリンク

作成
mklink /d d:\tmp\folder_aa c:\tmp\folder_aa

削除
rmdir d:\tmp\folder_aa

エラーへの対応

現在、指定されたネットワーク フォルダは異なるユーザー名とパスワードを 使って割り当てられています。 異なるユーザー名とパスワードを使って接続するには、 最初に現在のネットワーク共有の割り当てをすべて切断してください

net use ¥¥192.168.0.200 /delete

net use
