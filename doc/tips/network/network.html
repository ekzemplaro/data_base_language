network の設定 in Ubuntu 12.04 & 12.10

静的なアドレスの時の、DNS サーバーの設定

/etc/resolvconf/resolv.conf.d/head を編集。

nameserver 172.20.40.203
nameserver 172.20.40.204
nameserver 172.20.40.205
変更内容を反映

sudo resolvconf -u

DHCP サーバーからのアドレスの取得

sudo dhclient

Boot 時に、waiting for network configuration で止まった時
Alt + Ctrl + F2
