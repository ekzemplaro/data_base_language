Gentoo のインストール

# fdisk /dev/sda

# mkfs.ext4 /dev/sda1
# mkswap /dev/sda4
# swapon /dev/sda4

# mount /dev/sda1 /mnt/gentoo
# mkdir /mnt/gentoo/boot
# mkdir /mnt/gentoo/proc
# mount -t proc none /mnt/gentoo/proc

# cd /mnt/gentoo
# wget http://server/tmp/stage3-amd64-20140403.tar.bz2
# tar xvfj stage3-amd64-20140403.tar.bz2

# tar -xvjf portage-20140507.tar.bz2 -C /mnt/gentoo/usr

sshd の設定

手動で起動
/etc/init.d/sshd start

自動で起動
rc-update add sshd default

apache2 の設定

emerge apache

/etc/hosts の設定
172.20.208.93 host004
/etc/apache2/httpd.conf の設定
ServerName host004
apache2 の起動
/etc/init.d/apache2 start
/var/www/localhost/htdocs/index.html が表示される。
