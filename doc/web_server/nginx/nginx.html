Perl-FastCGI on Ubuntu

PHPを使えるようにする。

php5-fpm (FastCGI Process Manager) を使う。

参考になるページ

How to install Nginx, PHP5 / PHP-FPM, MySQL on Ubuntu 13.04

必要なパッケージのインストール

sudo apt-get install php5-cli php5-fpm php5-json

/etc/nginx/sites-available/default を使う方法

root /var/www; に変更する

スタートページは、/var/www/index.html になる。

#
	location ~ \.php$ {
	fastcgi_split_path_info ^(.+\.php)(/.+)$;
#
	fastcgi_pass unix:/var/run/php5-fpm.sock;
	fastcgi_index index.php;
#	include fastcgi_params;
	include fastcgi.conf;
	client_max_body_size 32M;
	try_files $uri =403;
	}
php5-fpm の再起動

sudo service php5-fpm restart

nginx の再起動

sudo service nginx restart

設定ファイルのチェック

sudo nginx -t

PHPで、アップロードできるファイルのサイズの上限を 512M にする。

/etc/nginx/sites-available/default

	location ~ \.php$ {
	fastcgi_split_path_info ^(.+\.php)(/.+)$;
#
	fastcgi_pass unix:/var/run/php5-fpm.sock;

	fastcgi_index index.php;
	include fastcgi_params;
	client_max_body_size 512M;
	}
/etc/php5/cgi/php.ini

post_max_size = 512M
upload_max_filesize = 512M
Perlを使えるようにする。

ii  fcgiwrap     1.0.3-3  amd64      simple server to run CGI applications over FastCGI
デフォールトで、

/var/run/fcgiwrap.socket が使われる。

/etc/nginx/sites-available/default への設定の追加

location ~ \.pl$ {
	gzip off;
	include fastcgi_params;
	fastcgi_pass unix:/var/run/fcgiwrap.socket;
	fastcgi_index index.pl;
	fastcgi_param SCRIPT_FILENAME /var/www$fastcgi_script_name;
	}
fcgiwrap の再起動

# service fcgiwrap stop
# service fcgiwrap start
# service nginx restart
