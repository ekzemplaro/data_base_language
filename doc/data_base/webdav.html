Windows の領域をWebDAV で、読み書きできるようにする方法

  1. fsmount というグループを作成する。
    2. sudo groupadd fsmount

  2. fsmount というグループで読み書きできるようにマウントする。
    3. sudo mount -t cifs -o "codepage=cp932,iocharset=utf8,
    username=user_aa,passwd=user_aa,workgroup=AAA,
    uid=nobody,gid=fsmount,file_mode=0775,dir_mode=0775"
    //host_windows/place /mnt/place

  3. apache2 を、fsmount というグループで実行する。
    4. /etc/apache2/httpd.conf

    Group fsmount

    sudo service apache2 stop
    sudo service apache2 start
