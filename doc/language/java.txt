<h2><a href="http://www.oracle.com/technetwork/java/index.html">Java</a></h2><p />
<blockquote>
<pre>
ii  openjdk-8-jdk:amd64   8u91-b14-0ubuntu4~16.04.1       amd64   OpenJDK Development Kit (JDK)
ii  libcommons-httpclient-java     3.1-12    all    Commons HTTPClient - Java library for creating HTTP clients
</pre>
</blockquote>

Ubuntu<br />
<blockquote>
	# apt-get install openjdk-8-jdk<br />
</blockquote>

Arch Linux<br />
<blockquote>
	# pacman -S jre8-openjdk<br />
	# pacman -S jdk8-openjdk<br />
</blockquote>

jar の中身の確認<br />
<blockquote>
	To show contents in aaa.jar<p />
	jar -tf aaa.jar<p />
</blockquote>

バージョンの確認<br />
<blockquote>
	javac -J-version<p />
	java -version<p />
</blockquote>

バージョンの切り替え<br />
<blockquote>
	sudo update-alternatives --config java<p />
</blockquote>

コンパイル<br />
<blockquote>
javac -encoding UTF-8 source.java
</blockquote>

実行<br />
<blockquote>
java -Dfile.encoding=UTF-8 source
</blockquote>

インストールされている Java 互換環境を表示<p />
<blockquote>
	$ archlinux-java status<p />
</blockquote>
